# cs1998_team21_backend

API doc:
Get all locations (GET/api/locations/)
Create a location (POST/api/locations/)
Get a specific location (GET/api/locations/{id}/)
Delete a specific location (DELETE/api/locations/{id}/)
Create a user (POST/api/users/)
Get a specific user  (GET/api/users/{id}/)
Add a user to a location (POST/api/locations/{id}/add/)
Create an vendor for a location (POST/api/locations/{id}/vendor/)
